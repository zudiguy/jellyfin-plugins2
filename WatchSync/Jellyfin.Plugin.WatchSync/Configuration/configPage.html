WatchSync

This plugin automatically synchronizes watch history between libraries of different qualities (4K/HD). When a movie is watched in 4K, the HD version is also marked as "watched" (and vice versa).

General

Enable or disable automatic synchronization during playback.

Completion Threshold

Media will be considered "watched" when this playback percentage is reached (default: 90%).

Data to Synchronize

Synchronize whether the media has been watched or not.
Synchronize playback position to resume where you left off.
Synchronize the counter of how many times the media has been watched.
Synchronize the date when the media was last watched.

Media Identification

Select which identifiers to use for matching media between libraries. At least one common identifier is required.

IMDB identifier (tt1234567). Very reliable for movies.
The Movie Database identifier. Reliable for movies and series.
TheTVDB identifier. Recommended for TV series.
Order in which identifiers are checked (comma-separated). The first match found is used.
If no identifier matches, attempt to match media by title and release year. Warning: may cause false positives.

Libraries

Uncheck libraries you want to exclude from synchronization.

Users

Uncheck users you want to exclude from synchronization.

Manual Synchronization

Run a full synchronization of all media for all users. Useful after first installation or to fix inconsistencies.
